# VHealth
一款基于天狗接口的健康资讯APP，主要分为以下几个功能：

1、健康和生活资讯浏览；

2、一些食谱、菜品的介绍；

3、可以查询一些疾病、药品、医院等信息；

4、美图浏览

5、健康书籍在线阅读（该部分还暂未实现）
